# streamblocks_2021

## 1 - server sur mac

* install :
  * git clone https://github.com/lesitevideo/streamblocks_2021/
  * cd streamblocks_2021
  * npm install


* usage : node server.js

## 2 - client sur raspberry

* install :
  * git clone https://github.com/lesitevideo/streamblocks_2021/
  * cd streamblocks_2021
  * npm install

* usage : node client.js | aplay -f S16_LE  -c1 -r 44100 -B 100000
